Forza Automation Assist Community Edition - Safety and Transparency

Community Edition includes:
- Auto1 Community
- Auto2 Navigation Test
- Auto3 Navigation Test

Forza Automation Assist does:
- Send normal keyboard input.
- Automate predefined FH6 flows through supervised desktop automation.
- Provide F8 emergency stop support.
- Operate externally as a Windows desktop application.

Forza Automation Assist does not:
- Modify FH6 game files.
- Inject into the game.
- Edit game memory.
- Access user accounts.
- Collect passwords.
- Require login credentials.
- Run automation on startup.

Operator responsibility
FH6 must be prepared at the documented baseline for the selected automation. Keep the visible game state under supervision and use F8 if the operation no longer matches the expected flow.

Windows security notices
Windows SmartScreen or antivirus software may warn because the application is not code-signed by a known publisher. Scan the downloaded ZIP and extracted folder if additional confidence is needed. Do not dismiss unexpected security warnings without reviewing them.

No absolute guarantee
This document describes the application's intended behavior. It does not make an absolute guarantee about game enforcement, antivirus classification, or every possible system configuration.

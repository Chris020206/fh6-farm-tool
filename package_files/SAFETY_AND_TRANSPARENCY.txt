Forza Automation Assist - Safety and Transparency

This file explains what the tool does and does not do.

The tool does:
- Send keyboard input.
- Automate predefined FH6 menu traversal and gameplay flows using keyboard input.
- Require (strongly recommends) user supervision during real-input automation.
- Provide F8 emergency stop support for guarded automation runs.

The tool does not:
- Modify FH6 game files.
- Inject into the game.
- Edit game memory.
- Bypass anti-cheat.
- Access user accounts.
- Collect passwords.
- Require login credentials.
- Run automation on startup.

Real-input behavior
Forza Automation Assist works by sending normal keyboard input through guarded automation paths. The operator remains responsible for starting from the correct FH6 baseline state and supervising the run.

Antivirus and SmartScreen
Because this is an early beta package from an "unknown publisher", Windows Defender SmartScreen or antivirus software may show a warning. That warning does not automatically mean the file is unsafe, but you should not ignore warnings blindly.

Recommended checks
- Scan the zip file before extracting if you want extra confidence.
- Scan the extracted folder before first launch if preferred.
- Ask through the Discord support route if anything looks unexpected.

No absolute safety claim
This beta is provided for controlled founding tester evaluation. It has validated behavior boundaries, but it is not public release software and should be supervised.

Forza Automation Assist - Known Issues

Build: v0.2.0-beta

Beta software
This is founding tester beta software. Packaging, onboarding, and desktop polish are still being validated.

Known limitations
- FH6 baseline state matters. Starting from the wrong in-game menu or screen can cause automation to desynchronize.
- Windows Defender SmartScreen or antivirus software may warn because the beta is not code-signed by a known publisher.
- Auto3 max validated cars = 4.
- Auto3 start row assumption = row A.
- Auto4 is not included.
- Unattended automation is not recommended.
- Packaging and onboarding are still under validation.

Automation risk notes
- Auto2 purchase mode can spend in-game credits.
- Auto3 unlock mode can spend skill points.
- Test modes should be used first when navigation confidence is uncertain.

Support
Report issues through the project Discord support route.

When reporting an issue, include:
- Which automation was selected.
- Whether the run was test mode or spending/unlock mode.
- What FH6 screen was visible before starting.
- What happened before the issue.
- Whether F8 stop was used.

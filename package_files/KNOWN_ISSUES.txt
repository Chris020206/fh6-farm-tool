Forza Automation Assist Community Edition - Known Issues

Version: v0.2.0-beta

Included automation
- Auto1 Community
- Auto2 Navigation Test
- Auto3 Navigation Test

Known limitations
- FH6 baseline state matters. Starting from the wrong in-game menu or screen can cause automation to desynchronize.
- Windows SmartScreen or antivirus software may warn because the application is not code-signed by a known publisher.
- Auto1 Community supports up to 5 consecutive race loops per execution and may be prepared again for later executions.
- Auto2 Navigation Test does not buy vehicles.
- Auto3 Navigation Test does not spend Skill Points.
- Auto4 is not included.
- Unattended automation is not recommended.

Support
Report issues through the official FAA Discord:
https://discord.gg/SgARD8YenU

When reporting an issue, include:
- The selected automation.
- The FH6 screen visible before starting.
- What happened before the issue.
- Whether F8 emergency stop was used.

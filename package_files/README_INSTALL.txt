Forza Automation Assist Community Edition - Install Guide

Forza Automation Assist Community Edition is a portable Windows application for guided, supervised FH6 automation. No installer or license import is required.

Included automation
- Auto1 Community
- Auto2 Navigation Test
- Auto3 Navigation Test

Installation
1. Download the complete ZIP package.
2. Extract the full folder before running the application.
3. Open the extracted folder.
4. Run Forza Automation Assist.exe.

Do not run the application directly from inside the ZIP file. The executable requires the accompanying folders and files.

Requirements
- Windows PC.
- Forza Horizon 6 installed separately.
- FH6 prepared at the correct documented baseline for the selected automation.
- Active supervision while automation is running.

Emergency stop
Keep F8 ready during automation. F8 is the emergency stop key for guarded real-input operation.

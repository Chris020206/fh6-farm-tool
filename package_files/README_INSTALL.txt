Forza Automation Assist - Install Guide

Beta status
This is a founding tester beta package. It is intended for controlled/manual evaluation, not public release.

What Forza Automation Assist is
Forza Automation Assist is a Windows desktop utility for supervised FH6 automation. It sends keyboard input to run Auto1, Auto2, and Auto3 automation flows while the operator remains present and supervises execution.

Portable app
This build is portable. It does not use an installer.

Recommended setup
1. Download the beta zip package.
2. Scan the zip file with your antivirus if you want to inspect it before extracting.
3. Extract the full folder before running the app.
4. Optionally scan the extracted folder.
5. Open the extracted folder.
6. Run:
   Forza Automation Assist.exe

Do not run the executable directly from inside the zip file.

Basic requirements
- Windows PC.
- Forza Horizon 6 installed separately.
- FH6 must be open and ready before automation is started.
- The correct FH6 baseline screen matters for each automation.
- The operator must supervise automation while it is running (or not, it is up to the user but we recommend active supervision).
- FH6 must be the active/focused window while automation is running.

Emergency stop
Keep F8 ready during automation. F8 is the emergency stop key for guarded real-input runs.

Support
Report beta issues through the project Discord support route. Include what you were doing, which automation was selected, and what FH6 screen was visible.

Important
This beta can send keyboard input and can spend in-game credits or skill points when purchase/unlock modes are used. Use test modes first when uncertain.

Licensing Documentation

This directory contains the design documentation for the FAA licensing system.

Documents are ordered as follows:

1. Licensing Core Architecture (Parts 1–6)
2. Feature Gating & Edition Logic (Part 7)
3. Final Licensing System Architecture & Roadmap (Parts 1–9)

The Final Licensing System Architecture & Roadmap is the primary implementation reference.

The earlier documents are preserved to document the evolution of the architecture and provide additional rationale where needed.

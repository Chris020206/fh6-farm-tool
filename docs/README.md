# Forza Automation Assist Documentation

This directory contains operational and engineering documentation for the
current controlled/manual MVP baseline.

The project is not public launch-ready and does not support unattended
automation. Auto1, Auto2, and Auto3 are validated for controlled/manual desktop
UI execution within their documented boundaries. Auto4 remains outside the
current implementation scope.

Primary source-of-truth documents:

- `VALIDATED_BEHAVIOR.md` - anti-regression baseline for Auto1, Auto2, and Auto3
- `PROJECT_STATE.md` - current strategic and technical project state
- `engineering_standards.md` - implementation boundaries and code standards
- `operator_runbook_auto1.md` - Auto1 operator procedure
- `operator_runbook_auto2.md` - Auto2 operator procedure
- `operator_runbook_auto3.md` - Auto3 operator procedure

The root `README.md` summarizes the repository structure, desktop UI status,
guarded input model, and validated automation boundaries.

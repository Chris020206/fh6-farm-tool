# R30 Home Composition Refinement

## Purpose

This checkpoint records Home screen composition refinement in the vertical companion prototype.

This is not a redesign.

This is not final styling.

## Goal

Improve the Home screen's opening composition, signal ordering, scan clarity, and feeling of:

> this is what matters now

while preserving:

- philosophy-first launchpad
- single-frame structure
- non-dashboard direction
- quiet confidence
- subtle guidance
- focused premium companion identity

## Refinements Applied

Home now emphasizes:

1. Recommended next step
2. Preparation for a supervised run
3. Lightweight recent context
4. Quiet controlled-MVP status

The primary action language was simplified from prototype-oriented wording to:

> Prepare Automation

This keeps the action focused on preparation rather than execution.

Home card titles were also quieted by removing structural role prefixes from visible card headings. The underlying metadata still preserves primary, secondary, and tertiary weighting.

## Composition Principle

Current principle:

> recommended next step first

Home should present a calm operational starting point rather than a dashboard, control panel, or activity feed.

## Preserved Direction

R30 preserves:

- vertical companion layout
- fixed window size
- hover overlay navigation
- stable main content
- no scrolling
- Home as philosophy-first launchpad
- Automation Environment as the preparation and commitment surface

## Non-Goals

R30 does not introduce:

- navigation behavior changes
- window size changes
- scrolling
- dashboard widgets
- execution wiring
- runner calls
- real input behavior
- automation logic changes
- safety gate changes
- `main.py` changes
- branding or logo work

## Current Status

The Home prototype should now feel more clearly ordered around what matters immediately.

Further refinement should evaluate:

- whether the Home signal count remains appropriate
- whether recent context deserves permanent Home presence
- whether the primary action treatment feels calm enough
- Automation Environment vertical readability

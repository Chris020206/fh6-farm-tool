# R20 Home Prototype Review

## Purpose

This document records what the R19 Home prototype taught us after translating product philosophy into an actual launch surface.

This milestone evaluates:

> whether Home behaves like the intended emotional and operational entry point of FH6 Farm Tool.

This is a review checkpoint.

No execution behavior, automation wiring, styling system, or production visual design was introduced in this milestone.

## Review Verdict

R19 Home Prototype is considered:

> Directionally approved.

The Home experience currently feels:

- Calm.
- Focused.
- Trust-building.
- Intentionally distinct from Automation Environment.
- Aligned with the current product trajectory.

The Home prototype should not yet be interpreted as final visual direction.

The current success signal is:

> product identity translation.

The prototype feels more like:

> FH6 Farm Tool beginning to feel like itself.

rather than:

> generic software scaffolding.

## What Worked

### Distinct Identity From Automation Environment

Home currently feels meaningfully different from the Automation Environment.

Automation Environment feels:

> commitment-oriented.

Home feels:

> orientation-oriented.

This separation reinforces:

> confidence formation before commitment.

and avoids collapsing the product into a single automation-launch surface.

### Philosophy-First Launchpad Direction

The current Home direction feels aligned with product philosophy.

Current trajectory:

> quiet launchpad + slight premium control-room feeling.

The prototype currently supports:

- clarity
- calmness
- intentionality
- refinement
- trust

without feeling:

- dashboard-heavy
- corporate
- administrative
- overwhelming

### Single-Frame Direction

The non-scrollable, single-frame direction currently feels appropriate.

Current intuition:

> if something important cannot fit inside one intentional frame, too much may be shown.

This currently aligns strongly with product philosophy.

Candidate principle retained:

> Home should remain non-scrollable unless a strong reason emerges.

### Lightweight Signals Avoid Dashboard Drift

The lightweight approach to:

- quick automation access
- relevant activity
- suggested next step
- quiet status

currently feels structurally appropriate.

The product currently avoids:

> information abundance pretending to be usefulness.

This is considered important for maintaining:

> quiet confidence.

### Subtle Guidance Preserved

The Home prototype preserved:

## Subtle Guidance

The software quietly guides without feeling controlling, assistant-like, or instructional.

The experience currently feels:

> presented with what matters now.

This reinforces:

- trust
- control
- clarity
- confidence

without introducing:

- productivity dashboard behavior
- assistant behavior
- excessive UI explanation

## Areas Still Requiring Investigation

### Exact Content Density

The precise amount of Home content still requires calibration.

Questions remain:

- How much recent activity belongs on Home?
- How dominant should quick automation access become?
- What deserves permanent presence?
- What belongs elsewhere?

Current direction:

> restraint before abundance.

### Sidebar Proportionality & Closure

Sidebar concerns remain unresolved.

Current observations:

- Sidebar feels vertically oversized relative to navigation density.
- Sidebar closure feels compositionally unresolved.

Future milestone should investigate:

> proportional refinement before behavioral cleverness.

### Final Visual Hierarchy

Visual hierarchy should not yet be over-judged.

Reason:

The prototype still lacks:

- meaningful visual refinement
- production spacing decisions
- mature content density
- styling system

Current evaluation remains:

> structural and philosophical.

## Preserved Home Principle

Home should remain:

> the strongest embodiment of FH6 Farm Tool philosophy.

If one screen reflects the product most clearly, it should be Home.

The opening experience should reinforce:

- calm confidence
- clarity
- intentionality
- refinement
- trust

before anything else.
